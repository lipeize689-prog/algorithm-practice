# 最大公约数与最小公倍数问题数学推导

## 问题描述

给定两个正整数 $x_0, y_0$，求有多少个正整数对 $(P, Q)$ 满足：

1. $\gcd(P, Q) = x_0$
2. $\mathrm{lcm}(P, Q) = y_0$

其中 $\gcd$ 表示最大公约数，$\mathrm{lcm}$ 表示最小公倍数。

---

## 第一步：变量替换

设：
$$
P = x_0 \cdot m, \quad Q = x_0 \cdot n
$$

其中 $m, n$ 为正整数。

由 $\gcd(P, Q) = x_0$ 可知：
$$
\gcd(m, n) = 1
$$
即 $m$ 和 $n$ 互质。

---

## 第二步：利用最小公倍数公式

最小公倍数公式：
$$
\mathrm{lcm}(P, Q) = \frac{P \cdot Q}{\gcd(P, Q)} = \frac{x_0 m \cdot x_0 n}{x_0} = x_0 \cdot m \cdot n
$$

已知 $\mathrm{lcm}(P, Q) = y_0$，因此：
$$
x_0 \cdot m \cdot n = y_0
$$

整理得：
$$
m \cdot n = \frac{y_0}{x_0}
$$

---

## 第三步：判断无解条件

令：
$$
k = \frac{y_0}{x_0}
$$

若 $y_0$ 不能被 $x_0$ 整除，则 $k$ 不是整数，无解，答案为 $0$。

若 $k$ 是整数，问题转化为：

> 求所有正整数对 $(m, n)$，满足：
> 1. $m \cdot n = k$
> 2. $\gcd(m, n) = 1$

---

## 第四步：互质与质因子分配

对 $k$ 进行质因数分解：
$$
k = p_1^{a_1} \cdot p_2^{a_2} \cdots p_t^{a_t}
$$

其中 $p_1, p_2, \dots, p_t$ 是互不相同的质数，$a_i \geq 1$。

条件 $\gcd(m, n) = 1$ 意味着 $m$ 和 $n$ **没有公共质因子**。

由于 $m \cdot n = k$，$k$ 的每个质因子 $p_i$ 的**全部幂次** $p_i^{a_i}$ 必须完整地分配给 $m$ 或完整地分配给 $n$，不能拆分。

---

## 第五步：计数

对于每个质因子 $p_i^{a_i}$：
- 要么全部给 $m$
- 要么全部给 $n$

因此每个质因子有 **2 种分配方式**。

共有 $t$ 个不同的质因子，所以分配方式的总数为：
$$
2^t
$$

每一种分配方式对应唯一一组 $(m, n)$，且自动满足 $\gcd(m, n) = 1$。

---

## 第六步：对应回 $(P, Q)$

由 $P = x_0 \cdot m$，$Q = x_0 \cdot n$，每一组 $(m, n)$ 对应一组 $(P, Q)$。

注意 $(m, n)$ 和 $(n, m)$ 在分配方式中是不同的（除非 $m = n$），因此答案就是 $2^t$。

---

## 第七步：特殊情况

- 若 $k = 1$（即 $x_0 = y_0$），则 $t = 0$，答案为 $2^0 = 1$，对应 $(P, Q) = (x_0, x_0)$。
- 若 $y_0$ 不能被 $x_0$ 整除，答案为 $0$。

---

## 第八步：算法流程

1. 输入 $x_0, y_0$。
2. 若 $y_0 \% x_0 \neq 0$，输出 $0$ 并结束。
3. 计算 $k = y_0 / x_0$。
4. 对 $k$ 进行质因数分解，统计不同质因子的个数 $t$。
5. 输出 $2^t$。

---

## 第九步：质因数分解算法的详细原理

### 9.1 核心定理

**定理**：若 $n$ 是合数（$n > 1$ 且不是质数），则它至少有一个质因子 $p \leq \sqrt{n}$。

**证明**：设 $n = a \times b$，其中 $1 < a \leq b$，则 $a^2 \leq a \times b = n$，所以 $a \leq \sqrt{n}$。$a$ 本身可能不是质数，但它的质因子一定 $\leq a \leq \sqrt{n}$。因此 $n$ 必有一个质因子 $\leq \sqrt{n}$。

### 9.2 标准实现代码

```cpp
int temp = k;  // 用临时变量存储待分解的数
int t = 0;     // 不同质因子个数

for (int i = 2; i * i <= temp; ++i) {
    if (temp % i == 0) {
        t++;                    // 发现一个新质因子
        while (temp % i == 0) {
            temp /= i;          // 除尽该质因子的所有幂次
        }
    }
}
if (temp > 1) t++;              // 处理剩余的大质因子

# 9.3 关于循环条件中 temp 的变化

常见疑问：for 循环的条件是 `i * i <= temp`，但循环体内会修改 temp，这样没问题吗？

**回答**：这正是算法的精妙之处。每次判断 `i * i <= temp` 时，使用的是当前最新的 temp 值。这样做有两个好处：

- **效率提升**：当 temp 被除尽小质因子后，数值变小，循环的上限 `√temp` 也会相应降低，避免不必要的迭代。
- **正确性保证**：根据核心定理，合数必有 `≤ √temp` 的质因子，因此算法不会遗漏任何因子。

**示例**：分解 `k = 60`

| 轮次 | i | 进入循环时 temp | i² ≤ temp? | 操作 | 循环后 temp |
|------|---|-----------------|------------|------|-------------|
| 1 | 2 | 60 | 4 ≤ 60 ✅ | 除尽因子2 | 15 |
| 2 | 3 | 15 | 9 ≤ 15 ✅ | 除尽因子3 | 5 |
| 3 | 4 | 5 | 16 ≤ 5 ❌ | 循环结束 | 5 |

最后 temp = 5 > 1，记录质因子 5。

# 9.4 为什么循环结束后 temp > 1 一定是质数？

假设循环结束后 temp > 1 且是合数，那么根据核心定理，它必有一个质因子 `p ≤ √temp`。

由于 temp 在整个过程中只减不增，因此 `√temp ≤ √原k`。循环已经遍历了所有 `i ≤ √原k`，按理说应该会遇到这个质因子 p 并把它除尽，但事实上没有。矛盾。

因此，循环结束后 temp > 1 一定是质数。

# 9.5 为什么不会漏掉两个互质因子相乘的情况？

有人可能会担心：会不会出现 `i × i > temp` 但 temp 实际上是两个互质因子相乘的情况（例如 `temp = p × q`，其中 p 和 q 都是大于 `√temp` 的质数）？

**回答**：这种情况在数学上不可能发生。

如果 `temp = p × q` 且 `p ≤ q`，则 `p² ≤ p × q = temp`，所以 `p ≤ √temp`。

既然 `p ≤ √temp`，那么循环一定会遇到 `i = p`（或 p 的某个因子），从而发现并除尽 p。因此，循环结束后 temp 不可能还是两个大于 `√temp` 的质数相乘。

**举例**：`temp = 91 = 7 × 13`

- √91 ≈ 9.54
- 较小的因子 7 ≤ 9.54
- 循环到 i=7 时会发现 91 % 7 == 0，除尽后 temp = 13
- 循环结束后 temp = 13 > 1，记录质因子 13

# 9.6 为什么需要最后一行 if (temp > 1) t++？

这行代码处理的是所有大于 `√原k` 的质因子。

**场景 1**：k 本身是质数（如 `k = 17`）
- 循环中没有任何 temp % i == 0 成立
- 循环结束时 temp = 17 > 1
- 如果没有最后一行，就会漏掉这个质因子，导致答案少乘 2

**场景 2**：`k = p × q`，其中 `p < q` 且 p 较小（如 `k = 3 × 7 = 21`）
- i=3 时找到因子 3，除尽后 temp = 7
- 循环结束时 temp = 7 > 1
- 如果没有最后一行，就会漏掉质因子 7

**场景 3**：`k = p²`（如 `k = 25`）
- i=5 时找到因子 5，除尽后 temp = 1
- 循环结束时 temp = 1，不需要记录

因此，最后一行是必须的，它捕获了所有在循环中未被除尽的最后一个质因子。

# 9.7 算法正确性总结

质因数分解算法基于以下数学事实：

- 任何合数都有不超过其平方根的质因子（保证循环能覆盖所有小质因子）
- 循环过程中不断缩小 temp，确保效率
- 循环结束后剩下的 temp 要么是 1，要么是单个质数（这是最后一个质因子）
- 最后一步 `if (temp > 1)` 确保这个质因子被统计

# 第十步：时间复杂度

质因数分解用试除法，时间复杂度 `O(√k)`。

由于 `y₀ ≤ 10⁵`，`k ≤ 10⁵`，`√k ≤ 316`，非常快。

# 第十一步：示例验证

输入：`3 60`

- `x₀ = 3, y₀ = 60`
- `k = 60 / 3 = 20 = 2² × 5`
- 不同质因子：{2, 5}，`t = 2`
- 答案：`2² = 4`

四种 (P, Q)：
- (3, 60)
- (15, 12)
- (12, 15)
- (60, 3)

与题目样例一致。

# 总结

核心思想是将问题转化为对 `k = y₀/x₀` 的质因子分配问题。由于互质条件强制每个质因子的幂次不可拆分，不同质因子的个数 t 直接决定答案为 `2ᵗ`。

质因数分解时，利用“合数必有不超过其平方根的质因子”这一定理，可以高效地分解。循环结束后剩余的大于 1 的数必定是质数，需要单独统计。
